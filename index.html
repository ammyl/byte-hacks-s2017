⚔ Code Quest ⚔

are you ready to go on a journey to learn code via adventure?

Code Quest is a platformer game that's geared to make learning code fun by having a female protag with her handy dandy laptop, going on a journey to different portals. Along the way, she'll bump into cats (based on octocat) that will share helpful lessons that'll allow her to solve the challenge that awaits at each portal.
